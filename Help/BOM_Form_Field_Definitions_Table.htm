BOM Form – Field Definitions Table

Field Name

Description

# of Units

Units per package.

Add / Change #

Radio buttons indicating the number of iterations BOM forms have undergone. Upon first save, the Add button is checked. Subsequent versions trigger the Change # button and incremental numbers in the text box. These fields are system-populated and require no user intervention, and the form must be in Final status for the tracking to take affect

Additional Information

Text box used to enter supplemental notes and information about BOM components.

Artist

Name of artist.  Read-only field taken from the First Name and Last/Group Name fields on the Selection screen

Date

Date form is created in mm/dd/yy format.  Default is current date but can be changed as needed.

Draft / Final

Radio buttons (Add/Change) and read-only text box (version number) which track the number of revisions form has undergone. Upon first save, Add button is enabled and version number is grayed out.  Subsequent saves trigger Change button and increment the version number text box. All fields are system-controlled, and form must be in Final status to activate this tracking.

Email

Text box listing e-mail address of user entering data.  Default entry is the current user (from the User Info screen) but can be changed as needed.

Imprint

The Logo Imprint used on the product; this is sometimes different than the label used to financially track to products.  Read only field from the selection screen.

Ink

Number of ink colors used in printing.

Label

Releasing label for financial puposes. Read only field from the selection screen.

Last Updated

Read-only field, populated automatically by the system, which displays the ID of user who last updated screen, along with time and date of update.

Part

Name of part to be included with finished product.

Phone

Text box listing telephone number of user entering data.  Default entry is taken from the User Info screen when the form is created, but it can be changed as needed.

Prefix/Local Prod #

The U.S. product number used to identify the selection.  Read only field from the selection screen.

Release Type

Pulldown field indicating whether release is produced for commercial or promotional release.  Read only field from the selection screen.

Releasing Family

Releasing Family.  Read only field from the selection screen.

Shrink Wrap

Checkbox used to indicate whether finished product is shrink wrapped.

Special Instructions

Text box for special instructions needed to make the product.

Spine Sticker

Checkbox used to indicate whether spine sticker is applied to finished product.

Street/Ship Date

Date the selection is released or shipped.  Read only field from the selection screen.

Submitted By

Name of user entering the information.  Default entry is current user but this entry can be changed as needed.

Supplier

Vendor supplying part.

Title

Release title.  Read only field from the selection screen.
