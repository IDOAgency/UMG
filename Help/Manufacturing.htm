Manufacturing

The Manufacturing screen is only available for editing to UML users. It provides supplemental fields, in addition to a subset of the data from the Selection screen.

Field Name

Description

Manufacturing Comments

A hyper link that opens a pop-up window in which comments may be entered

Distribution

Deep or hits

UML Contact

The UML contact user responsible for the release. This contact drop down list only displays users employed by the UML family

Units per Set

The number of units of the configuration to be produced. This is a read-only field from the PFM screen.

Vendor/Plant

The primary plant at which the product is to be manufactured

P.O. Qty

The amount to be produced by the primary plant

Exploded Total

The PO quantity times the number of units

Completed Qty

The actual number of manufactured product

Last Updated

User login ID of the user who entered or last updated the screen and the date the screen was last updated. Read-only field. The system automatically updates based upon user login ID.

To add a Vendor, the UML users select the Add Vendor button on the lower right hand side of the screen.

To delete a vendor, the UML user should select the "del" hyperlink on the right side of the row to delete.
