Schedule

All tasks associated with manufacturing a release are assigned on the Schedule screen. The Schedule screen is also used to directly update task information on previously entered selections.

When the Schedule screen is first opened, the listing on the left-hand side of the screen will include selections for the company under which the user operates. The schedule data form (right-hand side of the screen) will either display information relative to the highlighted record in the listing on the left, or it will display information that pertains to the selection that was just created and saved.

The top portion of the schedule section displays a subset of the data entered in the Selection screen. The bottom section of the screen displays either the tasks assigned to the selection or the options available to assign tasks to a newly created selection.

There are a couple of items to note with regard to the display of tasks on the Schedule screen. Key tasks are displayed in a green font to stand out from the others, label key tasks, appear in bold green. Additionally, if a task is considered past due (no completion date or the status does not indicate "Done"), the background color of the Due Date field will turn pink. If the completion date is past the due date, the background of the completion date field turns pink.

Field Name

Description

Task Assignment

Name of a task (usually a short description of a task)

Wks (Weeks to Release)

Number of weeks or business days prior to release when the task must be completed in order to meet the street/ship date

Due

Date due. Calculated based upon the street/ship date and the weeks or days to release

Complete

Actual date the task was completed

Status

Optional field for status information specific to the task

Vendor

Vendor

Holiday Indicator

Between the Wks and Due columns is a thin column that will contain a one-character code to indicate if a due date falls on a designated non-work day (i.e. "H" for holiday or "I" for inventory).

Applying the Task Filter

The display of the Schedule screen can be changed to show only label or UML tasks. To filter the schedule screen by label or UML tasks, select the appropriate option from the filter drop-down list on the Schedule screen.

The three options are all tasks, only label tasks or only UML tasks.

When the filter is applied to a schedule, only the tasks meeting the criteria will appear.

A default filter can be set-up for each user. The filter can also be disabled for users so that the user would not have the ability to change it.
