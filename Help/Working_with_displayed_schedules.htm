Working with displayed schedules

Once schedules have been created and saved, you are given several options for configuring the display to your requirements.

·        You can filter displayed tasks by department

·        You can filter displayed tasks by owner

·        You can sort tasks using any of the displayed columns as sort key

Procedures for all these are given below.

Applying task filters

The display of the Schedule screen can be filtered by department and/or task owner. In the example above, the owner filter has been applied to show only label tasks. Follow these steps:

·        To filter by department, pull down the Department list and select the department by which you wish to filter your display.

·        To filter by owner, pull down the Owner list and select the task owner by which you wish to filter your display. Your choices include All, Only Label Tasks and Only UML Tasks for physical products, Only eCommerce Tasks for digital products..

The screen refreshes and displays only those tasks meeting your filter criteria.

Note

A default filter can be set-up for each user.  The filter can also be disabled for users if a user does not require the ability to change it. Contact your system administrator for details.

Sorting task displays

Any column heading within the schedule can be used to sort displayed tasks. In the example above, notice tasks are sorted by the Task Assignment column. Simply click the blue column heading hyperlink for the column by which you wish to sort. The screen refreshes and tasks are sorted as you specify. Default sort sequence is by WKS (Weeks to release) in descending  order.
