Getting started

IMPORTANT

Before you can log on to Milestone, you must be setup with a valid user ID and password. Contact your system administrator for assistance with either task.

Milestone opens as a web page in a browser window. Go to this page as you would any other internet or intranet page. Do one of the following:

·        Double click on the Milestone icon on your desktop. See Creating Desktop Icons, below for a procedure for creating this icon.

·        Open Internet Explorer (IE) and go to the intranet web address:

http://milestone.umusic.com

To do this, enter the above string into the Address box, then press Enter.

The Milestone login page (shown above) appears.

Hint

With the Milestone login page displayed, pull down the Favorite menu (from your Explorer window) and click Add to Favorites. A link to the login page is added to your Favorites list.

Getting in

Follow these steps to log on to Milestone:

1.      Enter your user ID in the User Login text box. This is usually the same ID you use to log onto the network when you start your computer.

2.      Enter your password in the Password text box.

3.      Click Login or press Enter. The application opens in the same browser window.

See also

Working with Your User Info, in this section, for procedures for changing your password.

Getting out

Follow these steps to log out:

1.      Make sure you’ve saved any entries you’ve made on Schedule, Selection, Manufacturing, PFM and/or BOM forms you wish to keep.

2.      In the Milestone application page, click the Logoff menu tab, as shown above. The Milestone Login page returns.
