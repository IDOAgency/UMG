Enterprise Reports

Enterprise Releases without Schedules

This report will list all Release that do not have Schedules applied, the are not task assigned to the selection.

The Releases without Releases sorts by Company, Street Date, Artist, Title, & Prefix/Selection ID.

Enterprise Past Due Releases

This report list all active releases that have a Street Date prior to a specified date cut off date (effective date). This report gives the user an idea of what releases are still active past a selected date range, a past or relatively recent date range should be entered. The user can then cancel or close the projects that are no longer active.

The Past Due Releases sorts by Family, Company, Street Date, Artist, Title, & Prefix/Selection ID.

Enterprise To Do List

This report lists tasks that are due between dates entered in the selection criteria.

The To Do List sorts by Family, Task Due Date, & Prefix/Selection Number, and Task Name.

Enterprise Release Schedule by Label

The Release Schedule by Label provides a listing of releases with summary information for each release by label. This report also separates the 'Full Length' configurations from the 'Single' configurations.

The Release Schedule by Label Report sorts by Label, 'Full Length', and then 'Single' releases.
