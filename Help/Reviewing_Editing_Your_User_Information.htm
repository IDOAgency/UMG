Reviewing/Editing Your User Information

 

The User Info screen contains information about your user ID, including Login Name, Passsword, User Name, Email Address, Phone ande Fax Number. Follow these steps to review and/or edit your information.

1.      From the Labels tab, click User Info. The User Info screen (shown above) appears.

2.      Make changes to fields as needed, including:

·        Password

·        User Name

·        Email

·        Phone

·        Fax Number

3.      Click Save. The screen refreshes and your changes are saved.

Changing your password

For security reasons, Password entries in the User Info screen are represented as black dots. To change your password, follow the steps listed above, replacing the existing entry with your new password. Notice as you enter your new password, letters again are represented as dots.

IMPORTANT

Be sure to enter you password exactly as you want it. Unlike most applications, you aren’t prompted in Milestone to enter your password again for confirmation.
