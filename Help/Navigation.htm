System Navigation

Tabs and Menus

Tabs, located across the top of the screen, provide access to the four Milestone modules. Simply click on a tab to open a module.

The menu bar below the tabs display options which are specific to the selected module. Click on one of the options to open that specific screen.

Screens

Milestone is divided into two sections on every screen:

  • Search results listing or notepad (left-hand side of the screen)

  • Data form (right-hand side of the screen)

The listing displayed in the search results section is determined by queries (searches) initiated by the user. The data form displays the corresponding detail associated with the highlighted item in the search results list.

Sizing

The "%" button, located just under the menu bar, on the left-hand side of the screen, changes the size of the search results section. There are two sizes:

  0%

The search results section will occupy approximately 1/3 of the full screen

 100%

Only the data form will be displayed

Tool Tip

A tool tip bar is available on the notepad. When hovering over the Artist, Title or Street Date while selections are displayed in the notepad, additional information regarding the selection will appear. The tool tip displays the configuration, sub configuration, prefix, selection ID and UPC code.

Conducting Searches

The Search button is located in the upper left-hand corner of the notepad.

When the Search button is activated [by clicking the search button or by pressing the control key (Ctrl) and the letter (Q)] a dialog box appears. To close the box without actually conducting a search, click on the Search button again.

The available search criteria differ depending upon the screen from which the search is initiated. When multiple search criteria are entered, an item must meet all criteria to be included in the results listing.

Partial names or numbers may be used when conducting a search. For example, if a search for artist name is conducted by entering the letter "V" as the search criteria, all artists with "V" contained within the name will be displayed in the search results listing.

An '*' or '%' key can be used to search for words that begin with or end with what was entered.

For example:

  •  If "xxx*" is entered, the search will return all records that begin with "xxx" for the field selected.

  •  If "*xxx" is entered, the search will return all records that end with "xxx" for the field selected.

  •  If "*xxx*" is entered, the search returns all records with "xxx" occurring anywhere in the string.

The search results list can be sorted by clicking on any of the column titles. In this example, the list can be sorted by Artist, Title, or Street Date (Str Dt).

To view the entire results set for a listing once again, click on the Search button, leave all fields blank, and click on Submit Search or press the Enter key.

When searching for a selection or schedule, records with a status of "cancelled" or "closed" are not included in the search. To conduct a search for closed or cancelled records, click within the check box next to "Include Closed/Cancel", located towards the bottom of the search dialog box.

Copy, Save, New, and Delete Buttons

The Copy, Save, New and Delete buttons, are available to users with editing security rights and only for those records that pertain to their specific company. A user can copy a record, save changes made to a record, create a new record or delete a record.

Comments

There are small icons found on several screens throughout Milestone that look like small blank sheets of paper. When clicked upon, these icons provide access to a comments text box. To enter comments that pertain to a particular record, click on the icon next to the object to which the comments are to be attached. If comments are already present, the icon will appear as a sheet of paper with text and a pencil.

Calendars and Date fields

All date fields in Milestone can be populated by using a pop-up calendar. To activate the pop-up calendar, simply double click the left mouse button within a date field. This allows the user to use a calendar to select a date. The pop-up calendar is provided for ease of looking up a date and entering it into the Milestone system.

Milestone accepts dates entered directly into the date fields with dashes (7-25-00), dots (7.25.00) or slashes (7/25/00) for date fields. Dates will only be displayed on the screen with slashes.

Shortcut Keyboard Commands

Ctrl Q Opens the Search dialog box, (Q for query.)

Ctrl S Acts as the "Save" key. When using this shortcut, all of your changes will be saved for the record selected.

Ctrl C Activates the Copy feature on the Selection screen. This shortcut only works on the Selection screen.
