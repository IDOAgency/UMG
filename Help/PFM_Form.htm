Product File Maintenance (PFM) Form

The PFM form is an electronic form that replaces the Microsoft Word document currently being utilized. Once data is entered into the form, the PFM form can be printed or sent as an e-mail attachment.

When the PFM screen is first opened, the listing on the left-hand side of the screen will include selections for the company under which the user operates. The PFM form (right-hand side of the screen) will display information relative to the highlighted record in the listing on the left.

Description

Modify

Indicates a modification has been made to the PFM form

Oper Co.

Operating Company. The code that identifies the treatment of royalties for this release. Can be an appropriate two character alphanumeric code. Since the OpCo. is uniquely linked to the Label , selection of athe incorrect label code, will result in sales' passing to the wrong royalty system. The selected label code must be set up in that OpCo. If uncertain of the correct OpCo, please consult the appropriate controller.

Prod. No.

Product Number. The selection number for this release (the middle 10 digits of the 12 digit UPC number for commercial releases) This number is entered on the selection screen in the prefix and selection ID fields. It is a read-only field.

Conf. Cd.

Configuration Code. Select from the drop-down the best description of this release. To determine the appropriate Configuration Code, contact UML Inventory Accounting (818-866-9115)

Modifier

Modifier. Used to determine price export and record club sales.

Title

The title of the release. Automatically populated from the selection screen.

Artist

The name of the principal Artist (last name, first name). The artist field on the PFM form defaults to display the last name followed by first name of the artist based on the data entered on the selection screen. Please refer to the RMS guidelines for rules regarding artist name data entry.

Title ID

The product number to which this product should be linked. For audio, this field may remain blank if this is a CD release; otherwise, enter the product number of the CD to which this product is linked. This number is entered on the selection screen. It is a read-only field.

Release Date

The in-store date of this release. This is entered on the selection screen. It is a read-only field.

Super Label

Super Label

label Co.

Label Code. The code of the label making this release. Since the OpCo is uniquely linked to the label, it is very important to select the correct label code and ensure that the label code is set up in the appropriate OpCo. If uncertain of athe correct label code, please consult the appropriate controller.

Co. Cd

Select from the drop-down the best description of this release. It is important that "9000" is used if Operating Company "PR" or "IS" are already selected AND if this is a distributed label release which should be excluded from the Royalty Systems.

Proj ID

 Project ID. The label copy (RMS) number associated with this release.

PO Merge Cd

This field should reflect the label making the release; otherwise, select "99: UMG Labels"

Units per Set

The number of discs or tapes per set. This appears as a read-only field on the selection screen's# of units field.

Sets per Crtn

Sets per Crtn. The number of sets shipped in each carton (30 is the standard for single unit albums in all configurations)

Suppliers

This field should reflect Cinram, Gloversville, Kings Mtn and/or Pinckneyville if this release is being manufactured at one of these suppliers. If the release is being manufactured at another facility, enter that supplier's numeric code. Contact UML Inventory Accounting (818-866-9115) to obtain the correct supplier code.

Imp. Ind

Import Indicator. Determines if the release is US Made or Imported.

UPC

Universal Product Code. This number is entered on the selection screen. It is read-only field.

Music LNn

Music Line. Select from the drop-down the best description of this release. If "9000" was indicated for the Co. Code, "9000" should also be indicated here. Contact UML Inventory Accounting (818-866-9115) to obtain the correct music line code.

Rep. Owner

Repertoire Owner. The best description for this release. If unsure, contact A&R Administration or Business Affairs.

Rep. Class

Repertoire Class

Rtrn/Scrap Co.

Return/Scrap Code. The description of the return policy for this release.

Exp. Ind.

Export Indicator. The exports that are allowed for this release (if"E" or "I" are selected, also completed the next field). Contact Business Affairs for appropriate indicator if uncertain.

Incd/Exd Cnty

Included or Excluded Countries. The three-digit numeric code for each of the countries for which exports are to be included or excluded, if applicable.

Spine Title

The title of the release, including "(ED)", "(EN)" or "(EX)", if applicable the system will automatically populate the first 37 characters using the information from the Selection screen's Title field.

Spine Artist

The name of the principal Artist (last name, first name). Please refer to the RMS guidelines for rules regarding Artist name data entry. The system will automatically populate the first 37 characters using information entered in the Artist field on the Selection screen.

Price Cd

Price Code. The code which best corresponds to the pricing for this release ("Release Code" should not be used). Consult the Internal Pricing Table for the most common codes; if unsure, contact Sales Administration (818-777-8817). This number is entered on the selection screen in the sell code field. It is a read-only field.

Guar Cd

Gaurantee Code. Check "All penalties & bonuses apply / N" if this release is NOT IMI / IMP exempt; check "No penalties and bonuses apply / Y" if this release is IMI / IMP exempt.

IMP Rate Cd

Select "FL: Default" unless another description applies to this release. This field will be used -- along with Config Code -- to determine the incentive and disincentive values (bonuses and penalties) to be applied to this release's sales.

Music Type

Select from the drop-down the best description of this release. This is used to determine which sales reps will have responsibility for marketing this product, a responsibility that is usually defined by genre. If unsure, contact Sales Administration (818-777-8817). This number defaults in from what is on the selection screen in the genre field, but it can be changed for this form.

NARM Extract Ind

The Narm Extract Indicator. The values are "Yes" or "No". Choose "Yes" or the default "No". Products that should always be set to "No":

Value added SKUs

Customer specific Prepack SKUs

Premium SKUs

Custom manufactured SKUs

There may be other types of products that should also be set to 'No'. The theory is that any SKU that is not to be made available to ALL customers should be flagged as 'N'.

Price Point

Select from the drop-down the best description of this release. If unsure, contact Sales Administration (818-777-8817)

Last Updated

User login ID of the user who entered or last updated the screen, and the date the screen was last updated. Read-only field. The system automatically updates based upon user login ID.

Entering PFM Data

Conduct a search to find the selection (refer to Conducting Searches in the System Navigation section of this document).

Click on the record in the search results list to access the appropriate information.

Enter data into the data form, using drop-down lists where available.

Click the Save button to save the PFM form.

PFM Copy Feature

Click on the 'Copy' paper icon above the "Prepared By" field.

  • The 'Copy' paper icon only appears on PFM forms that have been saved.

Conduct a search to find the PFM form to paste this information to by using the Selection Search.

Click on the "Paste" clipboard icon to paste information from the previous PFM form to the new PFM form. Click on the Save button to save the PFM form.

  • Only PFM forms that have not been previously saved will display the "Paste" clipboard icon.
