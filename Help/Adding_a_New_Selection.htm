Entering a New Selection

Click on the New button (just under the menu bar on the right-hand side of the screen). Enter data into the data form, using drop-down lists where available.

The following fields require completion prior to saving the selection record:

  • Project ID (unless the project is a P&D)

  • Artist

  • Title

  • Selection ID

  • Family

  • Company

  • Division

  • Label

  • Product Category

  • Release Type

  • Configuration

  • Subconfiguration

If a selection is created without a Selection or UPC number, Milestone generates a temporary number. This number consists of the letters MLST followed by a sequential number.

If only the UPC number has been entered for a release, Milestone will generate the selection number automatically form the UPC number by striping the first and last digit of the code.

Milestone will copy the selection number to the Title ID field for most configurations. If the configuration is a full length Cassette, Vinyl or an Enhanced CD, the system will ask the user if the Title ID number matches the selection number. If the user responds 'Yes', the selection number will be copied to the Title ID field. If the user responds 'No', the Title ID field needs to be filled in. The Title ID allows the user to link a selection to another selection with a different configuration. If you are not sure how to respond to the Title ID prompt, select yes.

The retail code and price are determined by the system once the user selects the sales code (Sell) and saves the information.

The "Last Updated" field reflects the last user to add or update a selection and is automatically updated by the system.

Click the Save button to save the new selection. Once the record is saved, Milestone will automatically navigate the user to the Schedule screen for assignment of a template (refer to Assigning Task by Applying Templates in the Schedule of this document).
