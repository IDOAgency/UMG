Working with your Outputs

Report outputs are generated and displayed in the Acrobat Reader, as shown above. Notice the toolbars at the top of this window. Use these buttons to perform a variety of handy functions, including

·        Scrolling through report outputs

·        Modifying output displays

·        Printing report outputs

·        Saving report outputs to disk

·        Sending report outputs via e-mail.

Procedures for all these options are given in the sections that follow.

Scrolling through report outputs

For multiple page outputs, a series of scroll buttons (shown) are available – and they look an awful lot like the buttons on your CD player, don’t they? From left to right, these buttons perform the following functions:

·        Go to first page.

·        Go to previous page.

·        Go to next page.

·        Go to last page.

These buttons also appear at the bottom (left) of the window, along with the page number and total number of pages. To scroll through your output display, simply click the button that takes you where you want to go.

Printing report outputs

Follow these steps to print your report outputs.

1.      Generate a report output. See the sections above for procedures.

2.      Click the Print button (shown) on the toolbar. The standard Windows print dialog box appears.

3.      Click the Fit to Page checkbox (IE 5.x), if needed, to ensure your report prints properly. A checkmark (Ö) must appear here for proper printing. 

4.      Click OK to print.

Note

For later Internet Explorer versions, the Fit to Page option appears as Shrink oversize pages to paper size. It still must be checked.

Saving report outputs to disk

Follow these steps to save your report outputs to disk in PDF format.

1.      Generate a report output. See the sections above for procedures.

2.      Click the Save button (shown) on the toolbar. The standard Windows save dialog box appears.

3.      Enter a name for your file, as needed.

4.      Navigate to the folder in which you wish to save your report.

5.      Click Save to complete the save operation. Your report is saved to the folder you specify.

Sending report outputs via e-mail.

Follow these steps to send your report outputs via e-mail.

1.      Generate the report and save a copy of it to disk. See the above procedures for details.

2.      Attach the report file to an e-mail and send it the way you normally would. Because of variations in Windows versions and e-mail programs, detailed procedures are not given here. Contact your local helpdesk if you need assistance attaching files and/or sending e-mails.
