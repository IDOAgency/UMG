Creating/Editing Bills of Materials (BOM)

1.      Click on the BOM menu tab. Bill of Materials screen appears.

2.      Using the Notepad, either scroll to or search for the product for which you wish to create a BOM.

3.      Click anywhere on the product title to select (highlight) it. The BOM screen for the selected product appears. Keep in mind the following points:

·        Selection entries (Artist, Title and so on) are read-only, go to the Selection screen if you need to change them.

·        User identifying field entries (Submitted By, E-mail, Phone) are automatically populated with information about the current user, but you can overwrite them as needed.

4.      Click Draft or Final.

Notes about Draft/Final options

·        If Final is enabled, the Save button changes to Save+Send. E-mail notifications are sent whenever you click this button. To avoid superfluous e-mail notifications, please use discretion when changing BOMs to Final status.

·        The Draft/Final option is disabled completely if a Local Product Number has not been generated or the Project ID is not valid (example: 0000-00000).

5.      Complete your Bill of Materials, making entries in fields in the lower portion of the screen. Use the pulldown lists to make entries, where available.

6.      Click the Save or Save+Send (if final) button to save your BOM. The screen refreshes, the copy icon appears. Procedures for copying BOMs are given in the section that follows.

Copying Bills of Materials (BOMs)

Any saved Bill of Materials can be copied, then pasted into another BOM of the same foramt, provided it has not been saved. Follow these steps:

6.      Go to the BOM you wish to use as your original. See Creating BOMs, above, if you need to create one.

7.      Click the Copy icon (shown), upper left. The screen refreshes and BOM data is copied to your clipboard.

8.      Using the Notepad, scroll to or search for the BOM to which you wish to copy your original. The Paste icon should appear, upper left.

9.      Click the Paste icon (shown). The screen refreshes. BOM entries are pasted into the current screen.

10.  Make changes to fields as needed. Then click Save or Save+Send and continue.
