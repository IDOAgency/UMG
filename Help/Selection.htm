Selection

The Selection screen provides a data form in which information, specific to the manufacture of a product, is entered.

When the Selection screen is first opened, the listing on the notepad (left-hand side of the screen) will include selections for the company or companies under which the user operates. The selection data form (right-hand side of the screen) will display information relative to the highlighted record in the listing on the left.

Users will have security rights to view selections for their own company only. In addition, some users will have access to create and edit selections specific to an individual's own company.

FIELD NAME

DESCRIPTION

Street/Ship Date

Street date for commercial product or ship date for promotional product

Rls Wk/Cycle

The release week and cycle correlating to the street date

Int'l Date

International release date for the product

Impact Date

The radio impact or add date

Status

Status of the release (active, TBS, cancelled or closed)

Hold

A flag indicating a project is on hold

Hold Reason

A hyper link that opens a pop-up window in which information relating to why a project is on hold may be entered

P & D

A flag indicating that a project is a Press and Distribtion Deal

Special Pkg

A flag indicating special packaging is required for this project

Project ID

Project Id is a required field. The number entered here should be acquired from RMS (Label Copy). However, if the release is a P & D, then the Project ID will not be required.

Prefix

The prefix for promotional selections. This is filtered by company.

ID

The selection number for the collection of tracks or masters

Title ID

The product number to which this product should be linked. For audio, this field is always populated in Milestone if this is a CD release; otherwise, enter the product number of the CD with the same track list if there is one. This will be a read-only field populated by the selection number, unless it is a commercial full-length release. It will appear on the PFM form as a read-only field.

UPC

Universal Product Code

First

First name of the performing artist. Enter the artist name following standard RMS procedures.

Last

Last name of the performing artist. Enter the artist name following standard RMS procedures.

Title

Release title. Enter the title following standard RMS procedures.

A Side

The title of the first side or disc

B Side

The title of the second side or disc

Product Category

Category of the product being produced (box set, catalog, new release or soundtrack, etc)

Rls Type

Type of selection to be produced (commercial or promotional)

Config

Physical format (vinyl, compact disc, cassette, DVD, etc)

Sub

The sub-configuration (full length, single, maxi, etc)

Family

The Parent company (IGA, Island Def Jam, etc)

Company

"Roster" company (Geffen, A&M, Mercury, GRP, etc)

Divison

P & L entity (Decca, MCA Nashville, JAKE)

Label

Production entity (Republic, Booga Basement, Magic, etc)

Sell Filter

The sell code filter

Retail

The retail code which is determined by the the sell code selected.

Price

The Price of the product. This is determined by the Sell Code selected.

# of Units

Number of units per configuration for the selection. Read-only field from PFM.

Genre

Type of music (Blues, Classics, Jazz, etc.)

Pkg

Packaging specs. Double clicking on the field displays a pop-up text box where specific packaging details may be entered.

Label Contact

The user responsible for the release. The name of the user adding the selection is entered automatically into this field by the system when a new selection is created. It can be changed after the selection has been saved.

Other contact

This field is used for any additional contact, such as the name of a salesperson or an A&R rep.

Last Updated

User login ID of the user who entered or last updated the selection and the date the selection was last updated. Read-only field. The system automatically updates based upon user login ID.

Origin Date

The date the selection was added to Milestone

Last Street Date Update

The last date the street date was changed
