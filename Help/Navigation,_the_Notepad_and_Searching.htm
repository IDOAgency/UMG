Navigation, the Notepad and Searching

Selections in the Notepad are filtered according to the access level assigned to your user ID. When you log in, all selections for your companies are listed. At the bottom of the selection list is a line which indicates page number, number of pages and total number of records in the Notepad.

You have two options for locating specific selections:

·        You can use the navigation buttons/links to scroll to the selection.

·        You can use the Search function and find selections matching search targets you configure.

Procedures for both methods are discussed below.

Using navigation buttons can be effective if you’re looking through a limited number of selections, or you know exactly where your selection is located. The Search function, described below, can be used to locate selections more precisely.


Searching for Selections

The Milestone search function provides capabilities for locating products in the Notepad based on criteria you enter. You might, for example, search for all products for which Elton John is the artist. Follow these steps.

Entering search targets, Conducting searches

1.      Click Search, top left, in the Notepad. Selection Search box (left) appears.

2.      Enter search criteria. Enter text in the text boxes, use pulldown lists when available.

3.      Click Physical or Digital to limit results to either format. Or use the default entry, Both, to return both formats.

4.      Click Include Closed/Cancel to search for closed and cancelled records in your search results. . A checkmark (ü)  appears.

5.      Click Submit Search or press Enter.

 

NOTE

If you use Prefix/Local Prod # and/or UPC Code fields as search targets, these fields are included in the Notepad’s display of your results.

Two things to keep in mind:

·        For criteria entered as text into text boxes (as opposed to using pulldowns or checkboxes), your searches produce results if your target is found in any part of the field. That is, entering John, returns John, Johnson, Fitzjohn  and Fitzjohnson. See the examples below.

·        Computers take things literally. That is, if you enter Elton John, but the artist name has been entered as John, Elton (which it is), your search won’t return any selections. For this reason, you might enter Elton only, to ensure results.

·        You can also use wildcards: % to represent a single character, * to represent any number of characters. Using wildcards changes your search into a literal search. That is, entering John*, returns only entries beginning with the string, John (such as John, Johnny  and Johnson, but not Fitzjohn). See the examples below.

In summary, selections are listed in your Notepad. You can locate your selections using navigation buttons or the search function. Once you’ve found it, you can select it; and once selected you can view the various forms and screens associated with it.

Using Hover Help

Use hover help to display details about titles in the Notepad before you select them. Move your cursor over Artist, Title or Street Date field, a window appears which lists configuration, sub configuration, prefix/local prod #, and UPC code. sorting Notepad selections.

Sorting selections in Notepad

You can sort Notepad selections using any column as sort key. A blue triangle appears next to the label of the column used as the sort key, pointing up or down to indicate the sort sequence. Follow these steps:

1.      Simply click once on the heading of the column by which you wish to sort. The Notepad refreshes and selections are sorted by that column in ascending order (1 to 9, A to Z).

2.      Click the same heading again to sort in reverse, or descending order (9 to 1, Z to A). A blue triangle next to the heading points up or down, indicating ascending or descending order.

HINT

When sorting by Street Date, always first search for selections using Street Date as criteria. Otherwise, your search returns a cumbersome number of selections with n/a in the Street Date, indicating that products are in TBS (to be scheduled) status.

Searching for tasks and templates

   

IMPORTANT

Before you can search for templates or tasks you must first be given access to the Admin tab. Contact your system administrator if you need this access.

Follow these steps to search for templates and or tasks. Use the screenshots above for reference.

1.      Click the Admin tab.

2.      Click either Templates or Tasks in the menu bar. The appropriate search dialog appears. If you’re in the Schedule screen, click the Switch to Tasks button at the bottom of the Notepad.

3.      Enter your search criteria. Keep in mind your searches behave exactly like Selection searches. See the examples given in Searching for Selections, above.

4.      Click Submit Search or press Enter. Templates/tasks matching your search criteria are listed in the Notepad.

Hint

When search results for tasks are displayed, notice the navigation buttons found at the bottom of Selections and Templates results do not appear. To undo a tasks search and display all tasks, run another search without specifying any criteria.

Hiding/Unhiding the Notepad

Click the percent sign (%) button to hide or unhide the Notepad. Milestone screens, such as Schedule, Selections, PFM, BOM and/or Manufacturing are shown full screen when the Notepad is hidden.
