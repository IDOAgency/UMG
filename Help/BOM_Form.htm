Bill of Materials (BOM) Form

Milestone offers an electronic Bill of Materials form that replaces the Microsoft Word document currently being utilized.

When the BOM screen is first opened, the listing on the left-hand side of the screen will include selections for the company under which the user operates. The BOM form (right-hand side of the screen) will display information relative to the highlighted record in the listing on the left.

Entering BOM Data

Conduct a search to find the selection (refer to Conducting Searches in the System Navigation section of this document).

  • Click on the record in the search results list to access the appropriate information.

  • Enter data into the data form, using drop-down lists where available.

  • Click the Save button to save the BOM form.

BOM Default User Info

Information from your user profile, email address, phone number, and fax number, defaults into the corresponding fields on unsaved BOM forms. If the your user information is incorrect, your profile can be updated using the User Info screen.

  • Only BOM forms that have not been previously saved will display the default user info on the clipboard icon.

Printing a BOM

To print a BOM, save the BOM first. After saving the BOM, a PRINT TO PDF hyperlink will appear. Click on this hyperlink to open the BOM form.
