Field Definitions Table

Field Name

Description

# of Units

Text box used to indicate number of units per package, a read-only field taken from the corresponding field on the PFM screen.

Artist

Name of artist.  Read only field taken from the First Name and Last/Group Name fields on the the Selection screen.

Comments

Popup window triggered by paper icon hyperlink, a read-only field from the Selection screen which contains supplemental information about the product and/or form. Icon changes to paper with pencil when comments have been entered.

Completed Qty

Quantity, or number, of actual units manufactured.

Distribution

Pulldown list (Deep or Hits) used to determine the depot(s) from which products are shipped to fill sales orders, once they are available.

Exploded Total

Read only field calculated by multiplying PO Quantity by # of Units.

Hold

Checkbox indicating whether production for the release is on hold. Read only field taken from the field of the same name on the Selection form.

Hold Comments

Popup window, triggered by paper icon hyperlink, used to enter reason for hold. Read only field taken from the corresponding field on the Selection screen. Icon changes to paper with pencil when comments have been entered and saved.

Imprint

Logo appearing on product package, a read-only field taken from the corresponding field on the Selection screen.

Label

Releasing label for the product. Read-only field taken from the corrsesponding field on the Selection screen.

Last Updated

Read-only field, populated automatically by the system, which displays the ID of user who last updated the screen, along with time and date of update.

Manufacturing  Comments

Popup window, triggered by paper icon hyperlink, used to enter supplemental manufacturing information. Icon changes to paper with pencil when comments have been entered and saved.

P&D / Dist.

Checkbox indicating whether the release is associated with either a P&D (press and distribute) or Distributed (distribute only) deal. Read only field taken from the corresponding field on the Selection screen.

P.O. Qty

Requested manufacturing quantity.

Prefix/Local Prod#

Read only field, taken from the Selection screen, which displays U.S. product number for the selection. 

Project ID

Nine-character (nnnn-nnnnn) Project ID to which the selection belongs. Read-only field from the selection screen brought in from Archimedes when selection is created. If project ID is not available from Archimedes, 0000-00000 is used by default until valid ID is assigned.

Releasing Family

Releasing family, a read-only field taken from the corresponding field on the Selection screen.

Special Pkg

Checkbox indicating non-standard packaging (such as digi-pack) is being used for the product, a read-only field taken from the corresponding field on the Selection screen.

Status

Release status, a read-only field taken from the corresponding field on the Selection screen.

Street/Ship Date

Street date for commercial product, ship date for promotional product. Read only field taken from Street/Ship Date field on the Selection screen.

Title

Release title, a read-only field taken from the corresponding field on the Selection screen.

Type/Format

Read-only field that combines the Prod Category, Release Type, Format and Sub-Format fields from the Selection screen.

UML Contact

Pulldown list used to specify the UML contact responsible for the release.

UPC

Universal Product Code, a read-only field taken from the corresponding field on the Selection screen.

Vendor/Plant

Pulldown list used to specify the name of manufacturing facilities at which the release is to be produced.
