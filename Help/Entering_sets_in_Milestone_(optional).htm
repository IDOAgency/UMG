Entering sets in Milestone (optional)

Follow these steps to set up multiple unit sets, as in box sets or two-LP sets, in the Selection screen.

1.      Create and save a Selection screen for your set carrier, using either an auto-generated local product number or by entering a promo number. See the procedures above for details.

2.      Click on the UPC or Local Product # links. The component popup window appears (shown) when you click either link.

3.      Click Add. A blank line appears.

4.      Enter component’s Local Product Number.

5.      Enter component’s UPC in the text box provided.

6.      Enter a brief description of the component in the text box provided.

7.      Do one of the following:

·        Click the Add button again to enter another component. Repeat these steps until all components have been entered.

·        Click the Save button to save the component information you’ve entered. The Selection screen returns and both links (UPC and Local Product #) change to a purple color to indicate component information has been entered.

·        Click the del link to the left of any component you wish to remove.

8.      Click the Save button in the Selection screen to save the Selection record.

IMPORTANT

Even though you’ve clicked Save in the popup window, your component information is lost unless you save the Selection record. Thus, after entering and saving component information in the popup window, be sure to click the Save button from the Selection screen.
