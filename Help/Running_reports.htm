Running reports

Follow these steps to produce report outputs. Then see Working with your Outputs below, for procedures for saving, viewing, printing and sending reports via e-mail.

1.      Enter selection criteria in the data form (right-hand side of the screen) to filter data to appear on the report output.

2.      Click Save. The screen refreshes and the Print button appears.

3.      Click Print. A dialog box appears (shown) and prompts you to open the output or save it to disk.

4.      Click the Open the file… radio button.

5.      Click OK in the dialog box.

 

Note

The dialog shown appears for computers running Internet Explorer version 5.x. If the dialog box that opens looks different, you’re using a different version of Internet Explorer. The basic procedure is the same, you should be able to follow along without incident.

6.      If you are printing for the first time in Milestone and are using Internet Explorer 5.x, a second box identical to the first box will appear.  Uncheck the Always ask before opening this file type check box.

7.      Click OK. The report opens in Acrobat Reader.
