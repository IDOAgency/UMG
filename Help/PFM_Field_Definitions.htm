PFM Field Definitions

 

 

Field Name

P: Physical only

D: Digital only

B: Both

 

 

Description

 

# of Units

B

Text box used to indicate number of units per package.

 

Add / Change #

B

Radio buttons (Add/Change) and read-only text box (version number) which track the number of revisions form has undergone. Upon first save, Add button is enabled and version number is grayed out.  Subsequent saves trigger Change button and increment the version number text box. All fields are system-controlled, and form must be in Final status to activate this tracking.

 

Artist

B

Name of artist.  Read-only field taken from the First Name and Last/Group Name fields on the Selection screen.

 

Box Set

P

Checkbox indicating release is a box set.

 

Co. Cd

B

Pulldown list used to specify company code. Important: the code, 9000 Distributed Release, should be selected if Operating Company PR or IS is already selected (see above) AND release is from a distributed label release and should be excluded from royalties.

 

Comments

B

Popup window, triggered by paper icon hyperlink, used to enter supplemental information about the product and/or form. Icon changes to paper with pencil when comments have been entered and saved.

 

Comp Sndtrack

B

Compilation Soundtrack, pulldown list used to identify the type of soundtrack applicable to the release.

 

Config Cd

B

Read-only field on physical PFM, read/write on digital, imported from the Selection screen, which displays the Legacy Configuration Code used for the selection. 

 

Draft / Final

B

Radio buttons indicating form is in draft or final stage. Keep in mind automated email notification only applies to forms in Final status.

 

Email

B

Text box listing e-mail address of user entering data.  Default entry is the current user (from the User Info screen) but can be changed as needed.

 

Encryption

P

Pulldown list used to specify whether copyright protection encryption software is on the release.

 

Exp. Ind.

P

Export Indicator, pulldown list used to specify export guidelines for the release. Can be used in conjunction with Incd/Exd Cnty field, described below.

 

Fax Number

B

Text box containing the fax number of entering user. Default entry is taken from User Info screen, but can be changed as needed.

 

GRid #

 

D

Industry-wide number used to identify digital bundles.  Read only from the selection screen.

 

IMI Exempt

B

Pulldown list used to specify status of IMI penalties and bonuses for the release (N: All penalties & bonuses apply or Y: No penalties and bonuses apply).

 

IMP Rate Cd

P

Pulldown list used, along with Config Cd, to determine incentive and disincentive programs to be applied to sales orders for the release.

 

Imp. Ind

B

Radio buttons indicating whether release is imported (Import) or made domestically (US Made).

 

Imprint

D

The Logo Imprint used on the product; this is sometimes different than the label used to financially track to products.  Read only field from the selection screen.

 

Incd/Exd Cnty

P

Included/excluded countries, text box used to specify three-digit numeric codes for countries to be included or excluded from export, if applicable, used in conjunction with the Exp. Ind. field, described above.

 

Last Updated

B

Read-only field, populated automatically by the system, which displays the ID of user who last updated screen, along with time and date of update.

 

Loose Pick Exempt

B

Pulldown list used to indicate status of loose pick charges for the release (N: All loose pick charges apply or Y: All loose pick charges waived).

 

Modifier

P

Pulldown list which identifies the modifier code determining cost to export and record club.

 

Music Ln

B

Pull down list used to specify Music Line and its corresponding code. Keep in mind if 9000 is used for Company Code, 9000 should also be used here for Music Line.

 

Music Type

B

Pulldown list used to specify music type for the release, used to determine which UMVD sales reps have responsibility for marketing the product, a responsibility usually defined by genre.

 

NARM Extract Ind

B

Narm Extract Indicator, pulldown list used to specify whether release is included on the NARM extract. Current values are N:No and Y:Yes, default entry is N:No.

Set this indicator to N:No for any release which is not made available for sale to all customers, including:

  • Value added SKUs
  • Customer specific prepack SKUs
  • Premium SKUs
  • Custom manufactured SKUs

 

Oper Company

B

Read only field, imported from Selection screen, which displays code of Operating Company handling royalties for the product.

 

Parental Adv.

B

Parental Advisory, checkbox which indicates whether release contains explicit lyrics. This is a read-only field brought in from the Selection screen.

 

Phone

B

Text box listing telephone number of user entering data.  Default entry is taken from the User Info screen when the form is created, but it can be changed as needed.

 

PO Merge Cd

B

Pulldown list which reflects the label issuing the release. Notice selection, 99:UMG Labels, to be used for wholly owned US labels.

 

Prefix/Local Prod #

P

Read only field, imported from the Selection screen, which displays North America product number for the selection. 

 

Prepared By

B

Text box listing name of user creating the PFM.  Default entry is the current user (from the User Info screen), but can be changed as needed.

 

Price Cd

B

Price Code, read-only field, imported from Selection screen, which displays code used to apply pricing to the release.

 

Price Point

B

Pulldown list used to specify price point for the release.

 

Proj ID

B

Nine-character (nnnn-nnnnn) Project ID to which the selection belongs. Read-only field from the selection screen brought in from Archimedes when selection is created. If project ID is not available from Archimedes, 0000-00000 is used by default.

 

Release Date

B

Street date for commercial product, ship date for promotional product. Read only field taken from Street/Ship Date field on the Selection screen.

 

Rep. Class

B

Read-only field indicating Repertoire Class, populated automatically based on selection in Rep.Owner field.

 

Rep. Owner

B

Pulldown list used to specify Repertoire Owner for the release.

 

Rtrn/Scrap Cd

B

Pulldown list used to specify method for handling items returned to depots.

 

Sets per Crtn

B

Text box used to indicate standard carton quantity for the release/configuration.

 

SoundScan Grp

P

Read-only field, from the Selection screen, which displays SoundScan Group Code. This code groups different versions of the same releases for SoundScan reporting.

 

Spine Artist

P

Name of principal artist (last name, first name) to be printed on spine sticker of finished product. Default entry is populated automatically from the Artist field of the Selection screen, but can be changed here as needed.

 

Spine Title

P

Release title (including ED, EN or EX.) to be printed on the spine sticker of the finished product.  Default entry is populated automatically from the Title field of the Selection screen, but can be changed here as needed.

 

Sub Label

B

Read-only field displaying three-character Legacy sublabel code, imported from the Selection screen.

 

Super Label

B

Read-only field displaying two-character Legacy code for Super Label, imported from the Selection screen.

 

Suppliers

P

Pulldown list used to specify manufacturing supplier, such as Kings Mtn, Gloversville, Pinckneyville, Cinram and so on.

 

Title

B

Release title.  Read only field imported from the Selection screen.

 

Title ID

P

Read-only field used to group (in Legacy) different formats (CD, CT, Vinyl and so on) of identical releases.

 

UPC

B

Read-only field displaying Universal Product Code from the selection screen.

 

Value Added

P

Checkbox indicating value-added component (such as poster, CD, web link and so on), or anything that adds value to the standard release.
