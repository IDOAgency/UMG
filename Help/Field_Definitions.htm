Field Definitions

Field Name

Description

Task Assignment

Brief description of a task, entered by user defining it.

Wks (Weeks to Release)

Number of weeks or business days prior to release by which the task must be completed in order to meet the product’s street/ship date.

 

NOTE

Between the Wks and Due columns is a thin column that contains one-character codes to indicate if due date falls on a designated non-work day (H for holiday or I for inventory).

Due

Calculated date of expected completion for task, based on the product’s street/ship date and the weeks or days to release defined for tasks.

Complete

Actual date of completion for a task.

Status

Optional. Pulldown list used to specify various stages in task process.

Vendor

Optional. Text box used to enter names of vendors or contacts associated with tasks.
