Creating new product records

Follow these steps to create new product records:

1.      Click Selection in the menu bar under the Labels tab. (Typically, the Labels tab is activated by default when you log on.) The Selections screen appears.

2.      If the New button does not appear, you do not have edit access to the product you’ve selected. In this case, search for or navigate to a product for which you have edit rights. The New button should appear. See Basic Training for procedures for locating products.

3.      Click the New button (at the top of the Selection screen). The Project Search window appears. See Searching for Projects, below, to continue.

Searching for projects: the Project Search Window

Search  results for Benoit in the Last/Group Name field. Notice the unknown project  for the Verve label.

 

4.      Click either Physical or Digital radio button (at top) depending on the type of release you wish to enter.

5.      Enter search criteria using First Name, Last/Group Name, Project Title, Project Description or Project ID. You can enter partial search targets, if needed. Use the Label pulldown to search by Releasing Label.

6.      Click Search or press Enter to execute your search. Results are displayed in the list window at bottom. You can sort your results by clicking on column headings in blue. The blue triangle indicates which column is used as sort key and the sequence (ascending/descending) of the sort. Default sort is by Create Date, descending. That is, the most recently projects appear at top.

Note

For overall system performance, Milestone limits your project list to 200 projects. If your search returns more than 200 projects, a message appears at the top of the search window stating that only the first 200 are listed. If you see this message, enter more specific search criteria to limit your results to a more manageable number.

Or do one of the following:

·        Click Cancel to return to the Selection screen without specifying a project.

·        Click Clear to remove all entries and re-enter your search criteria.

7.      Double click the project you wish to use. The Selection screen – either physical or digital depending on your entries – returns. Project identifying data populates the appropriate fields. Notice some fields (Artist Name, Title and Imprint) can be overwritten; others are read-only.

Hint

An Unknown Project appears for every label family to which you have access. Use these unknown projects as placeholders for projects which haven’t yet been set up in Archimedes. Then use the Revise button in the Selection screen later, to import the appropriate project data – and overwrite the Unknown entries in Artist, Title, Description and Label fields –  when it becomes available.

Searching for UMVD projects (UML production users only)

UML production users must follow a slightly different procedure to locate projects for their products. Follow these steps:

·     Click the New button (at the top of the Selection screen). The Project Search window appears. By default, the Physical radio button is checked.

·     Enter your search criteria, using either the Label pulldown list or the Project ID text field.

·     Click Search or press Enter. Results appear at bottom.

·     Double click on the project you wish to use. Selection screen returns, project identifying fields are entered automatically.

NOTE – UML production users only

When searching by Project ID, omit the leading zero and the hyphen. Example: for Project ID 0069-1235, enter 0691235 in the Project ID field in the Project Search window.

Entering required fields

Fields with red asterisks next to them (an example is shown at left) must be entered before you can save the Selection screen. This section presents guidelines for completing these fields.

8.      Do one of the following:

·        For physical products, move cursor to Street/Ship Date field. Enter street date for commercial products, Ship date for promotional products.

·        For digital products, move cursor to Digital Release Date field. Enter date of digital release for the product.

·        Skip this step if the Status field is set to TBS (to be scheduled).

9.      Pull down the Status list and choose the appropriate status, if needed. Skip this step to use the default entry, Active. Choose TBS (to be scheduled) for product waiting for a date.

10.  Pull down the Releasing Family list and choose the appropriate entry if needed. Skip this step to use the default entry, which is the marketing label group assigned to your user ID.

11.  Move cursor to Imprint text box and overwrite as needed. Default entry is the label logo associated with the artist in Archimedes.

12.  Pull down the Product Category list. Choose the appropriate entry.

13.  Pull down the Release Type list. Choose Commercial or Promotional, depending on your product.

14.  Do one of the following:

·        For physical products, pull down the Format list and choose the appropriate entry.

·        For digital products, pull down the Schedule Type list and choose the appropriate entry.

15.  Pull down the Sub-Format list. Choose the appropriate entry.

NOTE:

·        At this point, digital products can be saved. To do this, simply click the Save button (there’s one at the top and bottom).

·        When creating physical products, complete as many remaining fields as needed for your product, then either a temporary product number or Local Product Number is required before you can save. See Generating Temporary Product Numbers and/or Generating Local Product Numbers in sections that follow.
